================================================================================
                    Red Insight - 项目目录结构
================================================================================

red-insight/
│
├── main.py                 # FastAPI 后端入口，API 路由定义
├── agent.py                # AI Agent 核心，意图理解与任务编排
├── scraper.py              # 小红书抓取模块，Playwright 浏览器自动化
├── logger.py               # 日志模块，多级别日志与文件轮转
├── config.py               # 配置文件，API Key 与模型设置
│
├── deploy.sh               # 一键部署脚本 (bash deploy.sh)
├── requirements.txt        # Python 依赖清单
├── README.md               # 项目说明文档
├── architecture.txt        # 本文件 - 目录结构说明
│
├── static/                 # 前端静态资源
│   └── index.html          # 单页面应用，对话界面 + 可视化报告
│
├── docker/                 # Docker 部署相关
│   ├── Dockerfile          # Docker 镜像构建文件
│   ├── docker-compose.yml  # Docker Compose 编排配置
│   ├── entrypoint.sh       # 容器启动入口脚本
│   ├── logrotate.conf      # 日志轮转配置
│   ├── README.md           # Docker 部署说明
│   ├── test_env.example    # 环境变量示例
│   └── scripts/            # 运维脚本
│       ├── archive_logs.sh # 日志归档脚本
│       └── setup_cron.sh   # 定时任务设置
│
├── docs/                   # 文档目录
│   └── ARCHITECTURE.md     # 详细架构设计文档
│
├── logs/                   # 运行时日志（自动生成）
│   ├── app.log             # 应用主日志
│   ├── error.log           # 错误日志
│   ├── requests.log        # API 请求日志
│   ├── agent.log           # Agent 执行日志
│   └── archive/            # 归档日志
│
└── venv/                   # Python 虚拟环境（自动生成）


================================================================================
                           核心模块关系
================================================================================

                        ┌─────────────────┐
                        │   index.html    │  前端界面
                        │  (静态页面)      │  对话 + 报告 + 执行日志
                        └────────┬────────┘
                                 │ HTTP API
                                 ▼
                        ┌─────────────────┐
                        │    main.py      │  FastAPI 服务
                        │  (API 网关)      │  路由 + 中间件 + 静态文件
                        └────────┬────────┘
                                 │
                                 ▼
                        ┌─────────────────┐
                        │    agent.py     │  AI Agent 核心
                        │  (智能体)        │  意图理解 + 任务编排
                        └───┬─────────┬───┘
                            │         │
               ┌────────────┘         └────────────┐
               ▼                                   ▼
    ┌─────────────────┐                 ┌─────────────────┐
    │   scraper.py    │                 │   LLM API       │
    │ (小红书抓取)     │                 │ (阿里云百炼)     │
    │  Playwright     │                 │  Qwen 模型      │
    └─────────────────┘                 └─────────────────┘


================================================================================
                           启动方式
================================================================================

  方式 1: 直接运行
  ────────────────
  $ python main.py
  
  方式 2: 部署脚本（本地）
  ────────────────────────
  $ bash deploy.sh local
  
  方式 3: 部署脚本（Docker）
  ─────────────────────────
  $ bash deploy.sh
  
  访问地址: http://localhost:8080


================================================================================
                           配置说明
================================================================================

  1. AI 模型配置 (config.py)
     ─────────────────────────
     OPENAI_API_KEY   = "你的API密钥"
     OPENAI_BASE_URL  = "https://dashscope.aliyuncs.com/compatible-mode/v1"
     OPENAI_MODEL     = "qwen-turbo"
  
  2. 小红书 Cookie (scraper.py)
     ───────────────────────────
     COOKIES = [
         {"name": "a1", "value": "xxx", ...},
         {"name": "web_session", "value": "xxx", ...},
     ]
